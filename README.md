# End-to-End-Modern-Data-Stack-Pipeline-Using-DBT-Snowflake-Airflow-AWS
Designed an end-to-end cloud ETL pipeline using DBT, Snowflake, Apache Airflow, and AWS. Ingested CSV data from Amazon S3, transformed and modeled data with DBT, and orchestrated workflows via Airflow. Implemented scheduling, data quality checks, error handling, and alerts using AWS SNS.
